How to Install

  1. Download the APK, OBB and patch OBB (TapTap versions don't need OBB).
  2. Install the APK on your Android device.

    • If blocked, enable "Install from Unknown Sources" in settings.
  3. Place the OBB and patch OBB files in the following folder: 
    Internal Storage/
    Android/obb/com.PigeonGames.Phigros/
    • If the folder doesn't exist, create it manually.
  4. Ensure the OBB filename matches the Version Code: 
    main.X.com.PigeonGames.Phigros.obb (Where the X is the version code)
    patch.X.com.PigeonGames.Phigros.obb (Same for patch OBB)

    Example: main.12.com.PigeonGames.Phigros.obb for Phigros v1.3.0 (Version Code 12)

  5. Launch the game and enjoy!
