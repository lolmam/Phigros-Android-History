Phigros 3.13.0 (126) TapTap

Released: 2025-04-18

  • Added Extra - Immaculėe Sekai Collection
    • Aphrodite's Child (short ver.)
    • DevIAtiOn (short ver.)
    • NO x
  • Other fixed bugs
    • Fixed a bug where when selecting Another Me with a lower sorting order as the background, it will be covered by Another Me with a higher sorting order when re-entering the player page
    • Fixed a bug where all the unlocked song covers are Luminescence when playing Chapter 8 in OS mode with Replay
    • Fixed a bug where the Result screen cannot be popped up when playing Crave Wave with Replay
    • Fixed a bug where the chapter cover is missing when Chapters 5, 6, and 7 is not clear
    • Fixed a bug where the Introduction as the Player background is missing when playing Phigros for the first time
    • Fixed a bug where the Ul text of the replay button is displayed abnormally
  • TapTap accounts now need to be verified with Chinese ID to login.
